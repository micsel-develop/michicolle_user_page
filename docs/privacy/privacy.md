# プライバシーポリシー

本アプリ「みちこれ!」（以下「本アプリ」）を運営する個人（以下「運営者」）は、ユーザーの個人情報の保護について以下の通りプライバシーポリシー（以下「本ポリシー」）を定めます。

### 1. 取得する情報および利用目的
運営者は、以下の目的のために必要な範囲でユーザーの情報を取得・利用します。

| 取得する情報 | 利用目的 |
| :--- | :--- |
| メールアドレス | アプリへのユーザー登録およびログイン認証のため |
| 位置情報 | 国道踏破状況の判定およびアプリ内機能の提供のため |
| 端末識別子 | アプリの利用状況の把握および認証・セキュリティ管理のため |

※収集したメールアドレスをユーザーへの連絡やマーケティング等の目的で使用することはありません。

### 2. 第三者提供の禁止
運営者は、法令に基づく場合等を除き、ユーザーの個人情報をユーザーの同意を得ることなく第三者に開示・提供いたしません。

### 3. 外部サービスの利用について
本アプリでは、アプリの機能提供および運用管理のため、以下の第三者提供サービスを利用しています。各サービスのデータ収集・処理については、それぞれのプライバシーポリシーをご確認ください。

*   **Google Firebase**
    *   利用目的：アカウント認証（Auth）、メッセージ通知、クラッシュ状況の統計収集
    *   [Google プライバシーポリシー](https://policies.google.com/privacy)
*   **SNSログイン機能 (Google / Apple / Facebook)**
    *   利用目的：各プラットフォームのアカウントを用いた簡便なログイン認証のため
*   **アプリストア (Google Play Store / Apple App Store)**
    *   利用目的：アプリ内課金および決済処理のため

### 4. 有料プラン（パスポートプラン）について
本アプリでは、以下の有料プランを提供しております。

*   **パスポートプラン（年額プラン）：** 1,000円/年（年1回の自動更新）
*   **パスポートプラン（月額プラン）：** 100円/月（月1回の自動更新）
    *   ※年額プランは、月額プランを1年間継続契約する場合と比較して、10ヶ月分の料金となるためお得です。

#### サブスクリプションの解約方法
有料プランからフリープランへ変更（サブスクリプションの解約）を行う場合は、以下の手順で設定アプリからお手続きください。

**【iOSの場合】**
1. iPhoneの「設定」アプリを開きます。
2. 画面上部にある「Apple ID（名前）」をタップします。
3. 「サブスクリプション」をタップします。
4. 一覧から「みちこれ!」を選択し、「サブスクリプションをキャンセルする」をタップしてください。

**【Androidの場合】**
1. Android端末で「Google Play ストア」アプリを開きます。
2. 右上のプロフィールアイコンをタップします。
3. 「お支払いと定期購入」＞「定期購入」をタップします。
4. 一覧から「みちこれ!」を選択し、「定期購入を解約」をタップしてください。

運営者は、必要に応じて本ポリシーを変更できるものとします。変更した場合は、アプリ内または本ページにて告知いたします。
